feather-mvc
===========

FeatherMvc is a pure php framework, the main purpose of the project is:

* Simplicity
* Light-weight
* Easy-config

## Usage

### Directory Structure

```
+ public
    | - .htaccess
    | - index.php
+ config
    | - config.ini.php
+ application/
    | - bootstrap.php
    | + controller
        | - IndexController.php
    | + template
        | + indexController
            | - indexAction.tpl
+ library
+ tests
```
